My project idea is creating a website for a salon business. This would include a user 
profile that can be logged into, or creating a new account. There also would be A
function to book a new appointment and to leave a review. 

The intended user of this system is someone who is looking to book appointments for 
a salon, the main demographic is women in their 20s and 30s. Another intended user
is the owners and employees of the salon to see what is being booked and create availabilities.
I want the website to have an elegant look. 

The users will be able to create, edit, and delete appointments. They also 
will be able to sign up for an account with my system so all their data is stored
and useable on multiple devices.
